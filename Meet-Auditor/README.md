# 🚀 プロジェクト名：Meet-Auditor（ミート・オーディター）

![alt text](image.png)

**〜 Google Meet完全同期型「AI会計監査・経営参謀」エージェント 〜**

## 💡 設定した課題とその背景

オンライン会議（Google Meet）の場で「この機材を買います」「この案件は〇〇円で進めてください」といった重要な意思決定（会計イベント）が行われても、それが会計システムやSalesforce、管理用のスプレッドシートに反映されるまでにはタイムラグがあります。

手動による再入力の手間が発生するだけでなく、人間の記憶頼みになるため「入力漏れ」や「二重入力」といったヒューマンエラーのリスクを常に抱えています。

## 🤖 AIエージェントの自律的な振る舞い（価値の中心）

本プロダクトは、単なる文字起こしツールや議事録作成支援ソフトではありません。会議の文脈を自律的に解釈し、**人間が指示を出す前に「会計処理のトリガー」を引き、各システムへ先回りしてデータを繋ぐ自律型エージェント**です。

1. **Google Meet への自律参加とリアルタイム解析**
エージェントが会議に自動で参加。音声データをリアルタイムでキャプチャし、テキスティング（構造化）を行います。
2. **Geminiによる「会計イベント」の自律検出・推論**
会話の流れから、「機材の購入決定（15万円）」や「新規案件の発注合意」といった、会社の財務に影響を与える重要な意思決定（会計イベント）のみをGeminiが自律的に検出・抽出します。
3. **周辺ビジネスツールとの自動同期・仕訳起票**
会議終了と同時に、Salesforceの商談フェーズを自動的に「受注」へと進行させます。同時に、会計用のGoogleスプレッドシートへ「仕訳のドラフト（下書き）」を自動で入力します。
4. **Chrome拡張機能による「人間への最終確認（Human-in-the-Loop）」**
経理・財務担当者がGoogle Chromeを開いた際、ブラウザ上にポップアップで「Meetでの合意に基づき、この仕訳（15万円のPC購入など）を確定しますか？」とプッシュ通知を表示。ワンクリックで本番反映できる状態を整えます。

## ⚙️ 技術スタックと選定理由（実装力）

実運用に耐えうる「リアルタイム性」と「セキュアなマルチプラットフォーム連携」を意識したアーキテクチャです。

* **インフラ（実行環境）：Google Kubernetes Engine (GKE)**
* *選定理由:* 会議の音声ストリーミングデータをリアルタイムかつ低レイテンシで並列処理するため、オートスケーリングと高可用性に優れたGKEを採用。


* **AI・音声処理：Speech-to-Text API & Gemini 1.5 Pro**
* *選定理由:* 会議特有の「長尺の文脈（コンテキスト）」や「複数人の発話」を正確に理解し、高度な会計判断（推論）を行うため、200万トークンの長大コンテキストを扱えるGemini 1.5 ProをVertex AI経由で利用。


* **Googleサービス連携：Google Meet API & Chrome Extension（Chrome 拡張機能）**
* *選定理由:* 会議システムとのシームレスな同期、および経理担当者の日常のブラウジング（業務フロー）を邪魔せずに通知を届けるため。
