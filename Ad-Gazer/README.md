# 🚀 プロジェクト名：Ad-Gazer（アド・ゲイザー）

![alt text](image.png)

**〜 YouTube・広告連動型「自律マーケティング最適化」エージェント 〜**

## 💡 設定した課題とその背景

デジタルマーケティングにおいて、自社のYouTube動画の反響、市場のトレンド、そしてGoogle広告の運用データを手動で分析し、それをSalesforce（CRM）の顧客セグメントへタイムリーに反映させるのは極めて困難です。

手動分析によるタイムラグによって、せっかくの「トレンドの波」や「顧客の関心の高まり」を逃してしまい、最適なタイミングでの広告運用やインサイドセールスのアプローチ（メール送信など）の機会損失が発生しています。

## 🤖 AIエージェントの自律的な振る舞い（価値の中心）

本プロダクトは、マーケティングデータの収集から分析、そして広告出稿と顧客アプローチの実行までを人間の手を介さずに24時間ノンストップで自律駆動する「AIマーケター」です。

1. **YouTube APIによる市場の24時間自律監視**
自社チャンネルの動画コメント、再生数の急上昇、さらには競合他社のトレンド動画の動きを常時モニタリングし、市場の熱量をリアルタイムに解析します。
2. **Geminiによる「トレンドとB2B顧客」のクロス推論**
動画のマルチモーダル解析（映像・テキスト）を通じて、Geminiが市場の潜在ニーズを推論。「現在、〇〇という技術キーワードの関心が高まっている。Salesforce内の『製造業』セグメントの顧客に刺さるはずだ」といった高度なターゲット選定を自律的に判断します。
3. **Google 広告のリアルタイム・チューニング**
判断に基づき、Google Ads APIを介してターゲットに合致する広告キーワードの設定や入札戦略を自律的に即座に最適化（チューニング）します。
4. **Salesforce・Gmailと連動した一斉インサイドセールス**
Salesforceから該当する顧客セグメント（例：製造業のリード）を自動抽出し、その関心を捉えるYouTube動画を埋め込んだインサイドセールスメール（Gmail）を自動生成。ターゲットに向けて一斉送信するマーケティングワークフローを自動で実行します。

## ⚙️ 技術スタックと選定理由（実装力）

Googleの強力なマーケティングAPI群とAIプラットフォームをシームレスに調律するアーキテクチャです。

* **実行環境：Cloud Run + Cloud Scheduler**
* *選定理由:* 定期的な市場監視（Cloud Schedulerによるトリガー）と、トレンド検知時の高負荷な一斉配信・広告最適化処理を、サーバレスで迅速かつスケーラブルに実行するため。


* **AIプラットフォーム：Gemini Enterprise Agent Platform（Vertex AI）**
* *選定理由:* YouTube動画（映像・音声）のマルチモーダルな文脈理解と、SNSコメントなどのテキスト感情解析を融合させ、エンタープライズ品質の正確なマーケティング戦略（推論）を導き出すため。


* **Googleサービス連携：YouTube Data API, Google Ads API, Gmail API**
* *選定理由:* 視聴データの取得から、広告運用の自動化、顧客へのダイレクトアプローチにいたるまで、Googleエコシステム内で最も安全かつ強力に自動化フローを構築するため。
