# 🚀 プロジェクト名：Auto-Pilot（オート・パイロット）

![alt text](image-1.png)

**〜 Google×Salesforce「自律型インサイドセールス」エージェント 〜**

## 💡 設定した課題とその背景

現代の営業マンは、Salesforceへの活動記録の入力、大量のメール返信、Google Meetの日程調整、そしてGoogle アナリティクスによる顧客行動のチェックなど、数多くの煩雑な事務作業やツール間の往復に追われています。

この「ツールの分断」と作業負荷により、営業マンが本来最も時間を割くべき「顧客との深い対話や本質的な提案活動」に集中できず、コア業務の時間が大幅に削られているという大きな課題があります。

## 🤖 AIエージェントの自律的な振る舞い（価値の中心）

本プロダクトは、コンシューマー向けとビジネス向けの双方に強力なGoogleサービスと、SalesforceのデータをAIが縦横無尽に繋ぎ、商談獲得から顧客フォローまでを裏側で自律的に完結させる「AIインサイドセールス」です。

1. **Google アナリティクス / 広告による「購買予兆」の自律検知**
「特定の重要顧客（企業）が、自社Webサイトの料金ページを短時間に何度も閲覧している」といった、売上に直結する重要な購買シグナルや予兆をデータから自律的に検知します。
2. **SalesforceデータとVertex AIによる過去履歴の文脈解析**
予兆を検知した企業のリード情報をSalesforceから自動で引き出し、**Vertex AI Vector Search**を用いて過去の商談履歴や議事録、顧客の関心事をセキュアかつ高度に多角分析します。
3. **Geminiによる「先回り提案メール」の自動起票**
分析した文脈に基づき、Geminiがその顧客の現在の関心に100%パーソナライズされた最適な提案メールの文面を自動で生成（起票）します。
4. **カレンダー連携による「Meet日程調整」の完全自動化**
営業マン自身の**Google カレンダー**の空き時間をエージェントが自動で算出し、顧客に**Google Meet**の候補リンクを内包したメールをGmail経由で自動送信（または下書き作成）し、人間が気づく前に商談の土台を整えます。

## ⚙️ 技術スタックと選定理由（実装力）

確実なトリガー検知と、エンタープライズの顧客データを安全かつスマートに扱うためのアーキテクチャです。

* **実行環境：Cloud Run（トリガーベースのワークフロー実行）**
* *選定理由:* 顧客のWeb行動（アナリティクス）などのイベントをトリガーとして、必要な時だけ瞬時に起動し、一連の連携ワークフローを低コストかつ高速に処理するサーバーレス環境として最適であるため。


* **AI・データ連携：Gemini API & Vertex AI Vector Search**
* *選定理由:* Salesforce内の機密性の高い顧客データを、外部に漏洩させることなく安全（セキュア）にベクトル化・検索し、Geminiの高度な思考力を用いて自然な提案文脈を導き出すため。


* **Googleサービス連携：Gmail API, Calendar API, Google Analytics Data API**
* *選定理由:* 行動データの抽出から、メール送信、カレンダーの空き時間枠の確保まで、日常の営業活動のインフラであるGoogle Workspaceをプログラムから100%制御するため。
