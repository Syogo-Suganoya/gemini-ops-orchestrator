2. 「Mental Sparring Partner（共感・メンタル・スパーラー）」実装プラン
コンセプト: 単に慰めるのではなく、あえて「問い」を投げかけ、ユーザーの思考の解像度を上げる対話エージェント。

AIエージェントの自律性:

会話の主導権: ユーザーが黙り込んだり、愚痴のループに陥ったりした際、Geminiが「別の視点（メタ認知）」を自律的に提示。

感情の可視化: 会話中の言葉のトーンからメンタル状態を数値化し、危険域に達した際に「今日はもう寝ましょう」とアプリを強制終了させる等の介入。

技術構成:

実行: Google Kubernetes Engine (GKE)。

リアルタイムな感情分析（Natural Language API）と、Geminiによる高度な推論を並列で高速に回すために利用。

AI: Gemini 1.5 Flash（低レイテンシでチャットのテンポを維持）+ Natural Language AI（感情・エンティティ解析）。

記憶保持: Vertex AI Vector Search（「3ヶ月前も同じことで悩んでいましたね」といった長期的な記憶に基づく対話）。

UI: Flutter（心の温度感を表すアニメーションや、ダークモードに特化したデザイン）。
