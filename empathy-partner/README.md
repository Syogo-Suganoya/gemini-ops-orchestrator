# 🚀 プロジェクト名：Mental-Sparring-Partner（メンタル・スパーリング・パートナー）

**〜 思考の解像度を上げる自律型「共感・メンタル・スパーラー」エージェント 〜**

## 💡 設定した課題とその背景

現代のストレス社会において、メンタルケアを目的としたAIチャットボットは数多く存在します。しかし、従来のツールの多くは「ユーザーの言葉をオウム返しする」「ただ慰める」といった受動的なアプローチに終始しています。

これでは、ユーザー自身が愚痴のループ（思考の堂々巡り）に陥ってしまった場合、根本的な原因の解決や、ネガティブな感情からの脱却に繋がらず、メンタルヘルスの本質的な改善にいたらないという課題があります。

## 🤖 AIエージェントの自律的な振る舞い（価値の中心）

本プロダクトは、単にユーザーに寄り添うだけの存在ではありません。**会話の主導権を自律的に握り、時にはあえて鋭い「問い」を投げかけることで、ユーザーにメタ認知（客観的な視点）を促し、思考の解像度を自律的に引き上げるメンタルの伴走者**です。

1. **会話主導権の自律的コントロールと「問い」の提示**
ユーザーが黙り込んだり、同じ不満のループを繰り返している状況をエージェントが自律的に検知。Geminiが「別の角度からその問題を見てみましょう」と、思考を揺さぶる視点やメタ認知を促す問いかけを自ら仕掛けます。
2. **リアルタイムな感情分析による「強制的介入」**
対話中の言葉のトーンやテキストから、ユーザーの精神状態を24時間リアルタイムに数値化（モニタリング）します。ストレスレベルや疲労度が危険域に達したとエージェントが判断した場合、「今日はもうこれ以上考えるのはやめて、寝ましょう」と、アプリを強制終了させる等のセーフティ介入を自律的に実行します。
3. **Vertex AI Vector Searchによる「長期記憶」に基づいた対話**
直近の会話だけでなく、過去数ヶ月にわたる対話履歴をベクトル化して保持。Geminiが「実は3ヶ月前も、同じシチュエントで悩んでいましたね。あの時と何が違うと感じますか？」といった、人間の親友のように長期的な文脈を捉えた自律的な推論・対話を行います。
4. **心の温度感を表現するインテリジェントUI**
Flutterで構築されたUIが、会話の進行や感情分析のスコアとリアルタイムに同期。画面の色彩やアニメーションが「ユーザーの心の温度感」を表現するように変化し、ダークモードに特化したデザインで深い自己対話を視覚的にもサポートします。

## ⚙️ 技術スタックと選定理由（実装力）

チャットの「テンポ（低レイテンシ）」を最優先しつつ、高度な感情パトロールと長期記憶を両立させる、実運用を見据えたアーキテクチャです。

* **実行環境：Google Kubernetes Engine (GKE)**
* *選定理由:* 音声やテキストのリアルタイムなストリーミング感情分析（Natural Language API）と、Geminiによる高度な思考プロンプトを高並列・低レイテンシで処理し、ユーザーにストレスを与えない会話の応答速度を維持するため。


* **AI・言語モデル：Gemini 1.5 Flash & Natural Language AI**
* *選定理由:* メンタルケアにおいてチャットの「テンポ感」は極めて重要であるため、超高速かつ軽量なGemini 1.5 Flashを採用。同時に、裏側でNatural Language AIを併走させることで、エンティティ解析と感情値の定量的パトロールを同時に実行するため。


* **記憶基盤：Vertex AI Vector Search**
* *選定理由:* 数ヶ月に及ぶ長期間のユーザーライフログや会話コンテキストを、安全かつ高速にセマンティック検索し、エージェントの「忘れられない記憶」として対話にブレイクダウンするため。
