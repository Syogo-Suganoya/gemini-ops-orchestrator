1. 「Legacy Agent（デジタル魂継承エージェント）」実装プラン
コンセプト: ユーザーのデジタル資産と言動を学習し、万が一の際に「意思」を代行するエージェント。

AIエージェントの自律性:

生存確認の自律実行: 一定期間ログインがない、または特定のカレンダー予定（通院など）に基づき、自ら「大丈夫ですか？」と確認。応答がない場合にフェーズを移行。

資産・情報の整理: 散らばったドキュメントから「遺族に伝えるべき重要情報」を自律的に選別。

技術構成:

実行: Cloud Run + Cloud Scheduler（定期的な死活監視トリガー）。

AI: Gemini 1.5 Pro（長期コンテキストを活用し、ユーザーの過去のメールや日記を分析して「人格」をエミュレート）、Vertex AI Search（膨大な個人データから必要な情報を即座に抽出）。

感情再現: Text-to-Speech API（本人の声に近いカスタムボイスでメッセージを生成）。

トリガー: Gmail API / Google Calendar API（ユーザーの活動履歴の監視）。

審査員へのアピール:

「死」という究極のプライバシーを扱うため、Vertex AIのデータプライバシー保護（学習に使われない設定など）の重要性を強調することで、実運用への配慮をアピール。
